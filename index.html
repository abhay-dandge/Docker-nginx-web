Introduction

NGINX is a powerful web server that can be easily deployed using Docker. In this guide, we will walk through the process of setting up NGINX in a Docker container.

Setup

To get started, you'll need to have Docker installed on your machine. Once you have Docker installed, follow these steps:

  1. Pull the NGINX image: docker pull nginx
  2. Run the NGINX container: docker run -d -p 80:80 nginx

Configuration

Configuring NGINX involves creating a configuration file. Here's a basic example:

server {
    listen 80;
    server_name localhost;
    
    location / {
        root /usr/share/nginx/html;
        index index.html index.htm;
    }
}

Save this file as nginx.conf and mount it when running the container:

docker run -d -p 80:80 -v /path/to/nginx.conf:/etc/nginx/nginx.conf nginx
